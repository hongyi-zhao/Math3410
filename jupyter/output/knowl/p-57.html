The moral of Theorem 2.2.5 is that one vector in a set is a linear combination of the others, we can remove it from the set without affecting the span. This suggests that we might want to look for the most “efficient” spanning sets – those in which no vector in the set can be written in terms of the others. Such sets are called linearly independent, and they are the subject of Section 2.3.
