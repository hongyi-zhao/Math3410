Note on notation: it is common useage to drop the usual parentheses of function notation when working with linear transformations, as long as this does not cause confusion. That is, one might write \(T\vec{v}\) instead of \(T(\vec{v})\text{,}\) but one should never write \(T\vec{v}+\vec{w}\) in place of \(T(\vec{v}+\vec{w})\text{,}\) for the same reason that one should never write \(2x+y\) in place of \(2(x+y)\text{.}\) Mathematicians often think of linear transformations in terms of matrix multiplication, which probably explains this notation to some extent.
